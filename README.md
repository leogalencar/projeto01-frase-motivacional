# projeto01-frase-motivacional
Projeto 01 do Módulo 01 do BootCamp Dev Jr.

# Projeto do Capítulo:
1. Criar um repositório no GitHub exclusivo para o projeto.
2. Criar um projeto no VSCode (pasta). 
3. Criar um arquivo chamado index.html.
4. Criar ou escolher uma frase motivacional dentro do index.html.
5. Fazer um commit com a mensagem: “frase motivacional 1”.
6. Fazer um push.
7. Criar ou escolher  uma segunda frase  motivacional dentro do index.html.
8. Fazer um commit com a mensagem: “frase motivacional 2”.
9. Fazer um push.
10. Criar ou escolher uma terceira frase  motivacional dentro do index.html.
11. Fazer um commit com a mensagem: “frase motivacional 3”.
12. Fazer um push.

# Requisitos de Aprovação do Projeto:
* Enviar o link do repositório público do projeto no GitHub.
* Repositório exclusivo para o projeto
* Arquivo index.html salvo no branch main.
* Três commits com as mensagens e conteúdos esperados.
